# Changelog

## v0.2.0 - 2026-04-08

- retrabalho grande da interface web e mobile, com fluxo de lista -> conversa -> detalhes no celular
- melhorias no layout das mensagens, grupos de sequencia, divisores de dia e marcador de mensagens nao lidas
- selecao multipla de mensagens com copiar, favoritar, encaminhar e apagar
- mencoes com `@`, destaque visual, aviso na lista de conversas e atalho para abrir o privado do usuario marcado
- autocomplete de participantes do grupo ao digitar `@`
- player de audio custom com waveform, velocidade de reproducao e preview melhorado de gravacao
- camera com preview no app, captura direta e anexos mais robustos
- reorganizacao das configuracoes, redefinicao para padrao, privacidade e preferencias visuais extras
- suporte local mais completo para contatos, grupos, IA assistente, favoritos, pins, arquivamento e undo toast
- ajustes na base de migracao para Supabase para refletir os novos defaults do front
